Hjälp och support

Den officiella dokumentationen för AnduinOS täcker ett brett spektrum av problem som du kan stöta på. Dessutom är AnduinOS:s forum för gemenskapen fulla med användare och utvecklare.

På Ask Ubuntu kan du söka efter och fråga om mycket vanliga problem. Lösningarna för nästan alla AnduinOS-problem är identiska med lösningarna för Ubuntu. Du kan helt lita på Ubuntus dokumentation och expertis för att lösa problem med AnduinOS.
